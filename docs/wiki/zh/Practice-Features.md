# ✍️ 专项练习 (Practice Features)

听多多不仅仅是一个播放器，它还内置了针对输入（Input）与输出（Output）的深度训练工具。

## 1. 听写练习 (Dictation)
想要检验自己的耳朵？
- **实时听写**：在工作区底部展开听写面板，随着音频输入文字。
- **智能评分**：提交后，系统会将您的文字与标准文本进行比对，用不同颜色标注错误，并给出百分比成绩。

## 2. 口语跟读与发音分析 (Speaking)
这是听多多的核心优势：
- **录音跟读**：选中一段文字或句子，点击录音按钮模仿原声。
- **音素级 AI 分析**：录音结束后，系统会分析您的每个发音细节。
- **可视化反馈**：通过 **红色/绿色** 精确标注读错或读准的音素（Phonemes），让您像看乐谱一样看出发音问题。
