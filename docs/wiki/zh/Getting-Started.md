# 🚀 入门指南

欢迎来到听多多的世界！在开始正式学习之前，您需要先准备好您的学习环境。

## 1. 登录与注册
首次启动程序时，您需要创建一个本地学习账号。这有助于系统记录您的学习进度、听写准确率以及跟读成绩。

*   **创建账号**：在登录界面点击 **“注册”** 按钮。输入您的用户名并设置一个简单的密码。
*   **本地存储**：请放心，您的账户数据和学习记录完全存储在您的本地计算机上，不会上传到任何服务器。

## 2. 切换与管理用户
听多多支持多用户使用。
*   **多用户模式**：在启动页或退出登录后，您可以点击不同的用户头像进入各自的学习空间。
*   **个性化**：每个用户都有独立的资料库和学习统计。
