# 📚 资料管理 (Library)

在听多多的“资料库”中，您可以轻松组织所有的听力素材。

## 1. 导入本地文件
如果您已经拥有音频文件（如 MP3, WAV 等）：
1. 点击左侧导航栏的 **“资料库”**。
2. 点击 **“＋ 添加本地文件”**。
3. 选择您电脑上的媒体文件。系统会将其加入您的学习列表。

## 2. URL 网页导入
想要直接学习网页上的视频或播客？
1. 点击 **“🔗 URL 导入”**。
2. 粘贴支持的链接（目前支持主流视频平台）。
3. 程序将自动下载并提取音频流，将其转换为可学习的格式。

## 3. 搜索与排序
当您的资料库变大时，可以使用顶部的搜索框快速定位素材，或者根据“最近添加”或“标题”进行排序。
